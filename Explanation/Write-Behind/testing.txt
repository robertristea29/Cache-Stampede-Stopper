WHAT HAPPENS BEHIND THE SCENES:
Timeline:
─────────────────────────────────────────────────

T=0s    User: npm run test:home
        API receives goal request

T=0.01s API: Updates Redis cache (instant)
        API: Queues message: {matchId:1, score_home:1, score_away:0}
        API: Returns 200 OK to user
        
        ← USER GOT RESPONSE (they're happy!)

T=0.05s Worker: BRPOP waiting on redis
        Redis: "Message arrived!"
        Worker: Wakes up, gets message

T=0.1s  Worker: Parses JSON
        Worker: Executes UPDATE on PostgreSQL
        
        ← DATABASE UPDATED (in background)

T=0.15s Worker: Loops, BRPOP waits again
        Queue now empty

Shutdown Procedure
Terminal 1 (API Server - npm run dev)
🛑 Graceful Shutdown:
   Press: Ctrl+C
   
   You'll see:
   ✅ Server stopped
   ✅ Nodemon cleaned up
   
   Then: Close with X (or leave open)


Terminal 2 (Worker - npm run worker)
🛑 Graceful Shutdown:
   Press: Ctrl+C
   
   You'll see:
   ✅ Worker shutting down gracefully...
   ✅ Redis connection closed
   
   (The worker.ts has a signal handler that does this cleanly)
   
   Then: Close with X (or leave open)


Terminal 3 (PowerShell - Test Commands)
🛑 No shutdown needed:
   This terminal just ran commands
   It's idle and safe to use again
   
   Option 1: Just leave it open for next commands
   Option 2: Close with X
